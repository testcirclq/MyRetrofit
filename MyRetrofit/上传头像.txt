由于上传头像比较难

所以使用的是@Part MultipartBody.Part file

首先API

【API】
@Description("修改头像")
public interface HeadApi   {
    @value
    final String url="api/user/personal/avatar";
    @Multipart
    @POST(url)
    public abstract  Observable<String> onPostman(@Part MultipartBody.Part file);

}

【Task】
然后执行的用 
    @Override
    public Observable<String> getObservable() {
        RequestBody requestBody = RequestBody.create(MediaType.parse("image/png"), file);
        MultipartBody.Part part = MultipartBody.Part.createFormData("avatar", file.getName(), requestBody);
        return  ((HeadApi) getBaseApi()).onPostman(part);
    }
